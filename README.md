## ğŸ”’ Privacy & Security

### Data Storage

- **100% Client-Side**: All data stored in your browser's localStorage
- **No Server**: No data transmitted to external servers
- **Private**: Only you can access your data on your device
- **No Tracking**: No analytics or user tracking

### What Data is Stored

- User credentials (email, password, name)
- Journal entries and history
- Journal goals and progress
- Safety plan content
- Intentions and reminders
- User preferences

### Data Management

- **Export**: Currently manual (copy/paste from browser)
- **Delete Account**: Clear browser data to remove all information
- **Backup**: Use browser export features or manually save content

### Security Considerations

âš ï¸ **Important Notes**:
- Passwords are stored in plain text in localStorage (not encrypted)
- Anyone with access to your device can potentially access the data
- This app is designed for personal wellness tracking, not clinical records
- For sensitive information, consider additional security measures

### Disclaimer

This application is a supplemental wellness tool and **NOT** a replacement for professional mental health care. If you're experiencing a mental health crisis, please:
- Call 988 (Suicide & Crisis Lifeline)
- Text HELLO to 741741 (Crisis Text Line)
- Call 911 for immediate emergencies
- Contact a mental health professional
